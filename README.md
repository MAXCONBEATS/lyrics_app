# Lyrics

**Lyrics** — кроссплатформенное приложение для хранения текстов песен с группировкой по альбомам, удобным редактором с метками секций и мгновенной отправкой в ваш Telegram‑канал.

## Возможности

- 📝 **Редактор с метками** – быстрая вставка `[Chorus]`, `[Verse]`, `[Bridge]` и других секций.
- 📁 **Альбомы** – объединяйте треки в альбомы, управляйте ими отдельно.
- 📤 **Telegram‑экспорт** – отправка одного трека или целого альбома в ваш канал одним нажатием.
- 🔍 **Фильтр «В альбоме»** – быстрый просмотр песен, привязанных к альбомам.
- 🎨 **Уникальный дизайн** – тёмная тема с градиентом, шрифт Accidental Presidency.
- 📱 **Кроссплатформенность** – Android и Windows (десктоп-версия для тестирования и использования на ПК).

## Технологии

- **Flutter** (Dart)
- **Riverpod** – управление состоянием
- **SQLite** (sqflite + sqflite_common_ffi) – локальное хранилище
- **Telegram Bot API** – отправка сообщений
- **SharedPreferences** – сохранение настроек

## 🔧 Настройка Telegram‑отправки

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Добавьте бота в свой канал (с правами администратора).
3. Узнайте ID канала (например, через [@getidsbot](https://t.me/getidsbot) или методом `getUpdates`).
4. В приложении откройте **Настройки Telegram** (шестерёнка), введите токен и ID канала (обычно начинается с `-100`), сохраните.
5. Теперь при нажатии на иконку отправки (`send`) рядом с треком или внутри редактора текст песни появится в канале.

> ⚠️ **Важно**: для отправки сообщений необходимо, чтобы на устройстве был включён **VPN с полным DNS-туннелем** (или перенаправлением всех запросов через VPN). Если в вашей сети блокируется `api.telegram.org`, без такой настройки отправка может не работать. Рекомендуем использовать VPN-клиенты с опцией «DNS через туннель» или «Override DNS».

## 🖊️ Использование

- **Главный экран** – все треки, сортировка по дате добавления (новые сверху).
- **Фильтр «В альбоме»** – показывает только песни, привязанные к альбому.
- **Редактор** – введите название, выберите альбом, нажмите на метку секции, чтобы вставить её в текст. Кнопка «Сохранить» в правом нижнем углу.
- **Альбомы** – создавайте альбомы, перемещайте в них треки, отправляйте альбом целиком.
