# Lyrics

**Lyrics** — кроссплатформенное приложение для хранения текстов песен с группировкой по альбомам, удобным редактором и вставкой структурных меток (Intro, Verse, Chorus, Bridge и т.д.).  
Создавайте треки, сортируйте их по дате, назначайте исполнителя и альбом, экспортируйте песни в JSON и делитесь ими.

## Возможности

- ✍️ **Редактор с метками секций** – быстрая вставка `[Chorus]`, `[Verse]` и других меток в текст.
- 📁 **Группировка по альбомам** – объединяйте треки в альбомы, управляйте ими.
- 🔍 **Фильтр «В альбоме»** – просмотр песен, привязанных к альбомам.
- 💾 **Экспорт/импорт** – сохраняйте отдельные песни или целые альбомы в JSON, импортируйте обратно.
- 🎤 **Кнопка «MAXCON»** – быстрая вставка имени исполнителя.
- 🎨 **Уникальный дизайн** – тёмная тема с градиентом, шрифт Accidental Presidency.
- 📱 **Кроссплатформенность** – работает на Android (основная цель) и Windows (тестирование/демо).

## Технологии

- **Flutter** (Dart) – фреймворк для UI
- **Riverpod** – управление состоянием
- **SQLite** (sqflite + sqflite_common_ffi) – локальное хранилище
- **file_picker / share_plus** – работа с файлами и экспорт
- **flutter_launcher_icons** – генерация иконок


## Структура проекта
```text
lib/
├── main.dart                   # Точка входа
├── app.dart                    # Настройка темы и маршрутизации
├── models/                     # Модели данных (Song, Album, Section)
├── services/                   # Работа с БД, экспорт/импорт
├── providers/                  # Провайдеры Riverpod
├── ui/
│   ├── screens/                # Экраны (TracksScreen, AlbumsScreen, SongEditor...)
│   └── widgets/                # Переиспользуемые виджеты
└── utils/                      # Константы, утилиты
```
